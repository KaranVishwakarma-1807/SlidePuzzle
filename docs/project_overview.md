# Project Overview

Slide Puzzle is an Android game developed in Kotlin using Android Studio.

## Goal
The goal of this project is to create a simple and interactive sliding puzzle game for mobile users.

## Development Details
- Built with Kotlin
- Designed using XML layouts
- Developed in Android Studio
- Version control managed with Git and GitHub

## Future Improvements
- Add difficulty levels
- Add timer and move counter
- Add sound effects
- Add leaderboard
